Tips

  • The editor title and chat box names can be changed.
  • Use the "+" buttons to add new boxes to the bottom of the page. You may need to scroll down to see them.
  • Use the "x" button underneath a chat box to close that box. Chat boxes cannot be recovered once closed!
  • Save an HTML copy of your chat log using the "S" button. Also download the CSS into the same folder by pressing "C" to view it in this style.
