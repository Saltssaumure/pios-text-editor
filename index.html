Help

  • The editor title is editable.
  • Names can be typed on the top line of the chat boxes.
  • Use the "x" button underneath a chat box to close that box. Chat boxes cannot be recovered once closed!

Sidebars

  • + : Add chat boxes to the bottom of the page. You may need to scroll down to see them.
  • o : Hide/unhide x buttons which remove chat boxes.
  • ? : Open this help menu.
  • S : Save an HTML copy of your chat log.
  • C : Download the CSS to view saved chat logs in the piOS style. Place this in the same folder as the HTML files.
